# Contributing to MISAKA Network

Thank you for contributing to MISAKA Network.

## Setup

Install Rust


curl https://sh.rustup.rs
 -sSf | sh


Clone repository


git clone https://github.com/MISAKA-BTC/MISAKA-devnet

cd MISAKA-devnet/misaka-core-rs-v9-stealth-hardening


Build


cargo build --release


## Development

Format


cargo fmt


Lint


cargo clippy


Tests


cargo test


## Pull Requests

1. Fork repository
2. Create branch
3. Commit changes
4. Open Pull Request

Example


git checkout -b feature/new-wallet
